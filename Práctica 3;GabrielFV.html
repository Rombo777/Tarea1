Texto de Cita

Podemos calcular la longitud de la hipotenusa por aplicación del teorema de Pitágoras:

La hipotenusa al cuadrado es igual a la suma de los cuadrados de los catetos.

Texto de Código

Copie el siguiente programa BASIC:

        
set cateto1=2
set cateto2=3
hipotenusa=sqrt(cateto1^2+cateto2^2)
print "Hipotenusa=",hipotenusa

Y ejecútelo escribiendo RUN.

El ordenador mostrará:

hipotenusa=3.605551275
